## Project Summary

This repository contains the implementation of my Final Year Project, **"Predicting Beta-Lactam Resistance in *Escherichia coli* Using a Protein Language Model."**
The project employs the pretrained ProtBERT protein language model to generate contextual embeddings from protein sequences, which are then used to train a Random Forest classifier for predicting beta-lactam resistance. 
The workflow includes data preprocessing, feature extraction, model training, performance evaluation, feature importance analysis and virtual mutation analysis. 
Developed entirely in Google Colab using Python, the proposed model achieved an **accuracy of 94%**, demonstrating the potential of combining protein language models with machine learning for antimicrobial resistance prediction.
